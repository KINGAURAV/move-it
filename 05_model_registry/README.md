Register your trained Random Forest model in the **Vayu Model Registry** to ensure version control and easy deployment to the ML Service.

---

## Quick Start

1. **Register Model:** From your training notebook, use the Vayu Model Registry client to upload your `.joblib` model file.
2. **Add Metadata:** Tag the model with relevant information (e.g., `algorithm: random_forest`).
3. **Verify:** Check the registry in the Vayu AI Studio UI to confirm your model is ready for deployment.
