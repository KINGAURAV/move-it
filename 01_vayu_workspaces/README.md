---

## 🚀 Get Started

1. **Create Workspace:** Open Vayu AI Studio and provision a workspace.
2. **Clone/Upload:** Bring the `move-it` repository into your workspace.
3. **Install Dependencies:**
   ```bash
   cd move-it
   pip install -r requirements.txt
   ```
